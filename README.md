# Smart Internship & Career Portal

Smart Internship & Career Portal is a modern single-page web application developed to help students discover internship opportunities and make informed career decisions. The platform provides an interactive and visually appealing interface with advanced features such as internship search and filtering, AI-based career recommendations, animated statistics, skills dashboards, testimonial sections, and responsive design.

The application allows users to browse internships across multiple domains, including Web Development, Java Development, Python Programming, and Artificial Intelligence. Users can view detailed internship information through interactive modal windows and apply for opportunities directly from the portal.

### Features

* Responsive Single Page Application (SPA)
* Full-screen Video Background Hero Section
* Internship Search and Category Filters
* Internship Details Modal Popups
* Apply Now Functionality
* AI Career Advisor
* Skills Progress Dashboard
* Animated Statistics Counters
* Testimonials Slider
* Dark/Light Mode Toggle
* Contact Form
* Mobile-Friendly Responsive Design

### Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Typed.js
* AOS Animation Library

### Objective

The objective of this project is to create an engaging platform that connects students with internship opportunities while demonstrating modern front-end web development skills, user interface design, and interactive web technologies.

### Outcome

The project successfully delivers a professional internship and career guidance portal that combines functionality, modern UI/UX design, and interactive features to enhance the user experience.
